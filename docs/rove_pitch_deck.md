## Slide 7 · THE DIAGNOSTIC MOAT — LH/FSH Strips

### Turn any ₹30 strip into a clinical-grade reading with your phone camera

**How the strip scanning works:**

```
┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐
│ 1. DIP   │────▶│ 2. SCAN  │────▶│ 3. READ  │────▶│ 4. FUSE  │
│          │     │          │     │          │     │          │
│ Urine    │     │ Phone    │     │ CV quant-│     │ Combine  │
│ strip    │     │ camera   │     │ ifies LH │     │ with CM  │
│ 5 min    │     │ capture  │     │ + FSH    │     │ + period │
└──────────┘     └──────────┘     └──────────┘     └──────────┘
```

**CV Pipeline:**
1. Strip contour detection
2. Control line colour calibration (normalise across lighting)
3. LH + FSH band ROI extraction
4. LAB colour space intensity quantification (0-100 scale)
5. LH:FSH ratio calculation

**Manual fallback:** Radio buttons (No line / Faint / Medium / Dark) for users without good camera conditions

**Why this is defensible:**
- **Data network effect** — More scans → better CV model → more accurate readings
- **No hardware needed** — Inito charges ₹4,999 for a reader device. We use the phone camera.
- **Algorithm IP** — Multi-signal fusion is proprietary
